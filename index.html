Analisa Perbandingan Volunteered Geographic Information (VGI) dengan Mental Maps

Saya Keenan dari departemen Geografi FMIPA Universitas Indonesia. Saya sedang meneliti kualitas informasi geografis yang dihasilkan melalui media kertas (mental maps) maupun komputer (VGI) untuk skripsi saya.

Saya membutuhkan responden untuk memberikan informasi terkait tempat tinggal sementara, dan tempat makan terfavorit di sekitar Kampus UI Depok.

Jika kamu merasa memenuhi persyaratan berikut,

  1. Mahasiswa UI atau alumni UI tidak lebih dari 1 tahun.
  2. Sedang tinggal secara sementara (baik kos, apartemen, kontrak, rumah kerabat, maupun pondok) di sekitar (bukan di dalam) kampus UI Depok.

dimohon kesediaannya untuk menjadi responden penelitian.
